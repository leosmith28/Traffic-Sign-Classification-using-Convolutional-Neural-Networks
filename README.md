# Traffic-Sign-Classification-using-Convolutional-Neural-Networks
This project builds and trains a Convolutional Neural Network (CNN) to classify 43 different traffic signs from images using TensorFlow and Keras. The dataset used is the German Traffic Sign Recognition Benchmark (GTSRB), pre-split into training, validation, and test sets. Images are preprocessed by converting them to grayscale and normalizing pixel values before being fed into the network. The CNN architecture is inspired by LeNet, consisting of two convolutional layers with max pooling, followed by fully connected dense layers and a softmax output layer for classification. Two model architectures are trained and compared over 15 epochs, with accuracy and loss curves plotted to evaluate performance.
